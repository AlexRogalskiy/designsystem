Heading Strategy

To allow for accessibility while maintaining flexibility in design, we have chosen to use the semantically correct header tags (H1, H2, H3, etc.) combined with styling classes. In other words, we do not style the HTML tags, but rather decorate the appropriate tag with the desired design.

<h1 class="h1">First Heading</h1>
<p class="lead-paragraph">Some leading text</p>

<h2 class="h3">Second Heading</h2>
<p>This one is an H2 with H3 styling.</p>
