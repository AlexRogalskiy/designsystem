Input-felt

Standard input

Default tar input hele bredden av containeren den ligger i og vises som et block-element.

Inline input

Input kan også settes til å være
--inline
og vil da vises som et inline-block element.

Input innbakt i tekst

Dersom input brukes som en del av en tekst brukes en variant uten borders
--text-like
Merk at siden layout overlates til applikasjonen har ikke dette feltet noen definert bredde, dette må derfor styres av konsumenten.
